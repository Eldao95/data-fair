## Créer son premier jeu éditable

Nous allons créer notre premier jeu éditable à partir de la page d'accueil

1. Cliquez sur *Créer un jeu de données*  
2. Choisissez **Editable**  
3. Renseignez un titre et créez votre jeu de données


![Membres](./images/lessons/contrib-02-rest-datasets-01.jpg)  
*Jeu de données éditable sans colonnes*

Cliquez sur le bouton "+" dans le schéma pour ajouter votre première colonne.  
Renseigné la clé de votre colonne ainsi que le type de votre.

![Membres](./images/lessons/contrib-02-rest-datasets-02.jpg)  
*Clé et type de la colonne*

Ajouter l'ensemble de vos colonnes et cliquer sur le bouton **Appliquer**

![Membres](./images/lessons/contrib-02-rest-datasets-03.jpg)  


## Ajouter des lignes

 Dans la section **Données** vous avez le tableau de vos données, celui-ci est vide pour le moment.  
 Cliquez sur le bouton "+" de cette section pour ajouter une ligne.  
 Le formulaire de saisie de votre jeu données apparait.  

 ![Membres](./images/lessons/contrib-02-rest-datasets-04.jpg)  
 *Vous pouvez ajouter 1 ligne à votre jeu de données avec cette méthode*  

 Vous pouvez importer plusieurs lignes d'un fichier si celui-ci possède exactement les même colonnes que votre jeu éditable.

## Paramètres avancés

Lorsque vous créer un jeu de données éditable, vous pouvez initier le schéma à l'aide d'un jeu de données existant sur la plateforme.  
Le schéma de ce jeu de données va être recopié dans votre jeu de données éditable, mais les données ne seront pas copiées.  

Vous pouvez également conserver l'historique des révisions des lignes de votre jeu données.  
Ainsi, vous pouvez visualiser les modifications réalisées sur chacune des lignes

 ![Membres](./images/lessons/contrib-02-rest-datasets-05.jpg)  


L'application formulaire de saisie permet d'exposer une interface de saisie sur des sites externes et vos visiteurs peuvent alimenter votre jeu de données.  

Le jeu de données éditable peut également être alimenter par API.
